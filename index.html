Frequently Asked Questions

What is the minimum attendance required at RTU colleges?

Rajasthan Technical University and most affiliated colleges require a minimum of 75% attendance in each subject to be eligible for semester examinations. Some colleges set a higher threshold of 80%. BunkWise uses 75% as the default target but you can adjust it to any value using the ✎ button on the attendance results page.

Does BunkWise work for SKIT, Poornima, VGU, Arya, and other RTU colleges?

Yes. The attendance calculator works for any college that uses a standard ERP portal to display attendance reports — including SKIT, Poornima University, VGU, Arya College, JECRC, and all other RTU-affiliated institutions. The RTU SGPA calculator works for any college whose result PDF comes from rtu.ac.in. The autonomous SGPA calculator is specifically built for SKIT Jaipur students whose results use the autonomous exam format.

Is my data safe? Is anything uploaded to a server?

Your attendance data and RTU result PDFs are processed entirely in your browser — nothing is uploaded to any BunkWise server. The only exception is the SKIT autonomous SGPA feature, which sends the text extracted from your PDF to Google's Gemini AI API for analysis. This data is processed in transit and not stored. After analysis it is saved in your browser's local storage for 24 hours so you can revisit your result, then it expires automatically.

How is SGPA calculated at RTU?

RTU SGPA = Sum of (Grade Point × Credits) for all subjects ÷ Total Credits. The grade point scale is: A++ = 10, A+ = 9, A = 8.5, B+ = 8, B = 7.5, C+ = 7, C = 6.5, D+ = 6, D = 5.5, E+ = 5, E = 4, F = 0. Note that grade A carries 8.5 points (not 8) — this is a common source of manual calculation errors that BunkWise handles automatically.

How do I get my RTU result PDF?

Go to rtu.sumsraj.com or rtu.ac.in and navigate to the Results section. Enter your roll number to load your result. When the result page opens, press Ctrl+P (Cmd+P on Mac), set the destination to "Save as PDF", and save the file. Upload that saved PDF to the SGPA Calculator on BunkWise.

How many AI analyses can I do per day for the SKIT autonomous result?

The SKIT autonomous result analysis uses Google Gemini AI, which is a paid service. To keep BunkWise free for everyone, there is a limit of 3 AI analyses per user per day. This limit resets at midnight IST. The regular RTU SGPA calculator (for rtu.ac.in PDFs) has no such limit — it runs entirely in your browser with no API calls.

Can I use BunkWise on my mobile phone?

Yes. BunkWise is fully responsive and works on all screen sizes. The attendance paste and upload features work on mobile. For the SGPA calculator on mobile, you can use your phone's file picker to select a saved PDF from your downloads folder. The Class Wise calculator also works on mobile with touch-friendly controls.

How do I convert SGPA to percentage for RTU?

The commonly used approximation for RTU is: Percentage = (SGPA − 0.5) × 10. For example, an SGPA of 8.2 ≈ 77%. However RTU does not publish an official formula for this conversion. For official purposes — job applications, higher studies — always use your official RTU mark-sheet and contact RTU for an equivalence certificate if required.
